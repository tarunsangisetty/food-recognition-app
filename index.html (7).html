// About the Project
HOW IT
WORKS

A two-stage deep learning pipeline: first classify the food, then segment it to estimate portion size and calories.

This project integrates two powerful models working in tandem. The Vision Transformer (ViT) classifies food with state-of-the-art accuracy across 251 categories, while Mask R-CNN segments each food item to estimate its portion size. Together, they automate the entire calorie tracking process from a single photo.

Caloric density values are standardized per 100g, and portion size from segmentation is used to calculate the final calorie estimate — no manual input required.

Processing Pipeline
📸
Image Input User uploads a food photo
🔍
Vision Transformer (ViT) Classifies food category from 251 classes
🎭
Mask R-CNN Segments food & estimates portion size
⚡
Calorie Calculation Maps portion × caloric density → kcal
📊
Result Output Food name + confidence + calories
// Core Features
WHAT IT
CAN DO
🧠
ViT Classification

Uses a Vision Transformer to classify food items into 251 categories with 88% Top-3 accuracy, leveraging global attention mechanisms for superior recognition.

🎭
Mask R-CNN Segmentation

Precisely segments individual food items within an image, enabling accurate portion size estimation with IoU scores above 75%.

🔥
Auto Calorie Estimation

Automatically calculates caloric content using standardized nutritional data and the estimated portion size — no manual input needed.

📸
Single Image Input

Everything from a single photograph. No reference objects, no weight scale, no barcode scanning required.

🍽️
251 Food Categories

Trained on the FoodX-251 benchmark dataset covering a wide variety of global cuisines and food types.

📈
Portion Size Grading

Segments map to portion size categories (small, medium, large) that scale the caloric estimate appropriately for real-world accuracy.
